## üéØ Customization Guide

### Update Colors
Edit the Tailwind config in `index.html`:
```javascript
tailwind.config = {
    theme: {
        extend: {
            colors: {
                'groq-orange': '#DC2626', // Red - Primary color
                'groq-dark': '#0F0F0F',
                'groq-gray': '#1A1A1A',
            }
        }
    }
}
```

### Update Dates
Modify timeline dates in the Timeline section of `index.html` and update the countdown timer in `js/main.js`:
```javascript
const eventDate = new Date('2025-10-06T23:59:59').getTime();
```
