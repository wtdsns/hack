### Method 2: Manual Command Line

```bash
# 1. Initialize and commit
git init
git add .
git commit -m "Initial commit: Groq Hackathon Landing Page"

# 2. Create repository on GitHub at: https://github.com/new
#    Name it: groq-hackathon-landing

# 3. Push to GitHub (replace YOUR_USERNAME)
git remote add origin https://github.com/YOUR_USERNAME/groq-hackathon-landing.git
git branch -M main
git push -u origin main
```

### Method 3: GitHub Web Interface (No Git!)

1. Create new repo: https://github.com/new
2. Click "uploading an existing file"
3. Drag all project files
4. Commit changes
